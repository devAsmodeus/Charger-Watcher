# Charger Watcher — юридический пакет

Папка содержит обязательные документы для платного Telegram-бота @chargerwatcherbot, монетизируемого через Telegram Stars и работающего с пользователями из Республики Беларусь.

## Документы

| Файл | Назначение |
|---|---|
| [terms_of_service.md](terms_of_service.md) | Пользовательское соглашение: описание сервиса, тарифы, SLA, ограничение ответственности, прекращение, применимое право. |
| [refund_policy.md](refund_policy.md) | Политика возвратов Telegram Stars: основания, исключения, сроки, механика `payments.refundStarsCharge`. |
| [privacy_audit.md](privacy_audit.md) | Аудит текущей команды `/privacy` против Закона РБ № 99-З + drop-in замена текста. |

## Открытые решения, требующие внимания Оператора

- **Применимое право (ToS, п. 10).** В тексте по умолчанию указано законодательство Республики Беларусь — заменить, если Оператор фактически проживает/налогово-резидентен в иной юрисдикции.
- **Контакты ответственного лица в политике конфиденциальности.** Заполнить плейсхолдеры `[USERNAME_TO_FILL]` (Telegram) и `[EMAIL_TO_FILL]` (e-mail) в drop-in тексте `/privacy`.
- **Дата вступления в силу.** Во всех трёх документах стоит `[YYYY-MM-DD]` — заменить на дату публикации перед интеграцией.

## Порядок интеграции

1. Оператор заполняет плейсхолдеры (юрисдикция, контакты, дата).
2. Оркестратор подменяет тело `cmd_privacy` на drop-in текст.
3. Добавляются команды `/refund`, `/export_me`, `/terms`, `/refund_policy` (отдельные задачи).
4. Документы публикуются в репозитории и/или на статической странице, ссылка проставляется в `/start`.
